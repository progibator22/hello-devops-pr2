Отчёт по выполнению задания:

В ходе работы был создан репозиторий в GitHub. От актуальной ветки main была создана ветка feature/TASK-123, в которой добавлен новый файл с указанием ФИО и группы, а также выполнены ещё два дополнительных коммита с изменениями.

Параллельно от ветки main была создана ветка bugfix/TASK-3321. В ней внесены изменения в тот же файл, что и в ветке feature/TASK-123, но в других строках, после чего также выполнены два коммита.

Находясь в ветке bugfix/TASK-3321, были начаты новые изменения без их фиксации в коммите. Эти изменения были временно сохранены с помощью команды git stash. После этого выполнено переключение на ветку feature/TASK-123, где были выполнены вспомогательные действия (просмотр файлов). Затем выполнен возврат в ветку bugfix/TASK-3321 и восстановление ранее сохранённых изменений с помощью git stash pop.

Далее обе ветки (feature/TASK-123 и bugfix/TASK-3321) были отправлены в удалённый репозиторий. Локально выполнено слияние ветки feature/TASK-123 в main. При попытке слияния ветки bugfix/TASK-3321 в main возник конфликт, который был разрешён вручную. После разрешения конфликта изменения были запушены в ветку main.

В удалённом репозитории был создан пул-реквест для ветки bugfix/TASK-3321, в котором описаны внесённые изменения и процесс разрешения конфликтов. В пул-реквест были добавлены reviewers. После одобрения пул-реквест был успешно слит в ветку main через веб-интерфейс GitHub.
